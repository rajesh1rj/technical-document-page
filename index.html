Introduction

Html is Standard for Hyper Text Markup Language.which is the most widely used language on web to develop web pages.

  • HTML describes the structure of a Web page.HTML describes the structure of a Web page and consists of a series of elements.
  • HTML elements tell the browser how to display the content.
  • HTML tags label pieces of content such as "heading", "paragraph", "table", and so on.
  • Browsers do not display the HTML tags, but use them to render the content of the page.
Heading

The <h1> <h2> <h3> <h4> <h5> <h6> tags are used to define HTML headings.

  • <h1> defines the most important heading and important for SEO.
  • <h6> defines the least important heading.
Paragraph

  • The <p> tag is used to define the HTML paragraph element.
  • The paragraph element begins with the <p> tag and ends with the </p> tag.
  • The HTML paragraph element should not cotain tables and other block elements.

List

Html offers web authors three ways for specifying lists of information.All lists must contain one or more list elements.

  • (ul)- An unordered list. This will list items using plain bullets.
  • (ol)- An ordered list. This will use different schemes of numbers to list your items.
  • (dl)- A description list is a list of terms, with a description of each term.

Font

Font tag is having three attribute called size, color and face to customize your fonts. To change any of the font attribute at any time within your webpage.

Simply use the font tag.The text that follows will remain changed until you close with the font tag. You can change one or all of the font attribute with one font tag.

Note; The font and basefont tags are deprecated and it is supposed to be removed in a future version of HTML. So they should not be used rather,It's suggested to use CSS styles to manipulate your fonts.
